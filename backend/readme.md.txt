# Backend - Financial Goal Tracker

This folder contains the **Spring Boot backend** for the Financial Goal Tracker.

### Tech Stack
- Java 17+
- Spring Boot (REST API, Security, Data JPA)
- PostgreSQL/MySQL (database)
- JWT (authentication)

### Features (planned)
- User registration & authentication
- CRUD APIs for financial goals and transactions
- Progress tracking and reporting
- Scheduled jobs for recurring contributions
