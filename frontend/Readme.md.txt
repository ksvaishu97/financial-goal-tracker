# Frontend - Financial Goal Tracker

This folder contains the **Angular frontend** for the Financial Goal Tracker.

### Tech Stack
- Angular 16+
- Angular Material / PrimeNG (UI components)
- Chart.js / ngx-charts (visualizations)
- JWT-based auth (interceptor + guards)

### Features (planned)
- User login & registration pages
- Dashboard with list of goals
- Goal detail page with progress chart
- Add/update contributions and transactions
